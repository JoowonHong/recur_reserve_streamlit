# 🎬 촬영 예약 시스템 (Photography/Videography Reservation System)

Streamlit을 사용하여 만든 간단하고 직관적인 촬영 예약 시스템입니다.

## 🌟 주요 기능

- **시작 날짜/시간 선택**: 촬영 시작 날짜와 시간을 개별적으로 선택
- **종료 날짜/시간 선택**: 촬영 종료 날짜와 시간을 개별적으로 선택
- **다일간 촬영 지원**: 여러 날에 걸친 촬영 예약 가능
- **자동 촬영 시간 계산**: 시작과 종료 일시를 기반으로 총 촬영 시간 자동 계산 (일/시간/분 단위)
- **실시간 예약 요약**: 선택한 정보를 실시간으로 확인
- **직관적인 UI**: 깔끔하고 사용하기 쉬운 인터페이스

## 🚀 시작하기

### 필수 요구사항

- Python 3.8 이상 (권장: Python 3.10 또는 3.11)
- pip 패키지 관리자

### 설치 방법

1. **저장소 클론 또는 다운로드**

2. **필요한 패키지 설치**
```bash
pip install streamlit pandas
```

### 실행 방법

```bash
streamlit run app.py
```

실행 후 브라우저에서 자동으로 열립니다. (일반적으로 http://localhost:8501)

## 📦 의존성 패키지

- `streamlit`: 웹 애플리케이션 프레임워크
- `pandas`: 데이터 처리 (필요시 확장 가능)

## 🎨 화면 구성

### 메인 페이지
- **좌측**: 촬영 시작/종료 날짜 및 시간 선택 영역
- **우측**: 선택한 촬영 예약 정보 요약 및 총 촬영 시간 표시
- **사이드바**: 현재 선택된 시작/종료 일시 표시

### 기능 설명
1. **시작 날짜 선택**: 촬영 시작 날짜 선택 (오늘부터 90일 이내)
2. **시작 시간 선택**: 촬영 시작 시간 선택 (30분 간격, 00:00 ~ 23:30)
3. **종료 날짜 선택**: 촬영 종료 날짜 선택 (시작 날짜 이후)
4. **종료 시간 선택**: 촬영 종료 시간 선택 (같은 날인 경우 시작 시간 이후)
5. **자동 계산**: 선택한 일시를 기반으로 총 촬영 시간 자동 계산 및 표시 (일/시간/분)
6. **예약 확정**: 모든 정보 선택 후 확정 버튼 클릭

# 시간 슬롯 생성 (30분 간격)

    for minute in [0, 30]:  # [0, 15, 30, 45]로 변경하면 15분 간격
        time_slots.append(f"{hour:02d}:{minute:02d}")
