# 🎬 예약 시스템 (Recurring Reservation System)

Streamlit을 사용하여 만든 일반 예약 및 반복 예약 시스템입니다. 일회성 예약과 정기적인 반복 예약을 모두 지원합니다.

## 🌟 주요 기능

### 일반 예약 (app.py)
- **시작 날짜/시간 선택**: 예약 시작 날짜와 시간을 개별적으로 선택
- **종료 날짜/시간 선택**: 예약 종료 날짜와 시간을 개별적으로 선택
- **다일간 예약 지원**: 여러 날에 걸친 예약 가능
- **자동 예약 시간 계산**: 시작과 종료 일시를 기반으로 총 예약 시간 자동 계산 (일/시간/분 단위)
- **예약 관리**: 예약 생성, 조회, 삭제 기능
- **데이터베이스 저장**: SQLite를 사용한 예약 정보 영구 저장

### 스케줄 예약 (pages/Schedule.py)
- **반복 예약 설정**: 특정 요일에 정기적으로 반복되는 예약 설정 가능
- **스케줄 관리**: 스케줄 기반 자동 예약 생성 및 관리
- **시간 지정**: 예약의 시작 시간과 종료 시간을 자유롭게 설정
- **활성화/비활성화**: 반복 예약 활성화 및 비활성화 기능
- **자동 스케줄러**: APScheduler를 통한 자동화된 예약 생성

## 🚀 시작하기

### 필수 요구사항

- Python 3.8 이상 (권장: Python 3.10 또는 3.11)
- pip 패키지 관리자

### 설치 방법

1. **저장소 클론 또는 다운로드**

2. **필요한 패키지 설치**
```bash
pip install -r requirements.txt
```

또는 수동으로 설치:
```bash
pip install streamlit pandas APScheduler
```

### 실행 방법

메인 앱 실행:
```bash
streamlit run app.py
```

실행 후 브라우저에서 자동으로 열립니다. (일반적으로 http://localhost:8501)

Streamlit 멀티페이지 앱이므로 좌측 사이드바에서 "Schedule" 페이지로 이동하여 스케줄 예약 기능을 사용할 수 있습니다.

## 📦 의존성 패키지

- `streamlit` (≥1.28.0): 웹 애플리케이션 프레임워크
- `pandas` (≥2.0.0): 데이터 처리 및 분석
- `APScheduler` (≥3.10.0): 스케줄 예약 자동화

## 🎨 화면 구성

### 메인 페이지 (app.py)
- **일반 예약 폼**: 시작/종료 날짜 및 시간 선택 영역
- **예약 목록**: 생성된 모든 예약 표시 및 관리
- **예약 삭제**: 개별 예약 삭제 기능
- **예약 정보 저장**: SQLite 데이터베이스에 자동 저장

### 스케줄 페이지 (pages/Schedule.py)
- **스케줄 생성**: 반복 예약 설정 양식
  - 스케줄 이름 입력
  - 반복 요일 선택 (월~일)
  - 스케줄 시작/종료 날짜 설정
  - 예약 시작/종료 시간 설정
- **스케줄 목록**: 모든 스케줄 표시
- **스케줄 관리**: 활성화/비활성화, 삭제 기능
- **생성된 예약**: 스케줄에서 자동 생성된 예약 조회

## 💾 데이터베이스

본 프로젝트는 SQLite를 사용합니다:

- `reservations.db`: 일반 예약 저장
  - `reservations` 테이블: 개별 예약 정보
  - `repeat_groups` 테이블: 반복 예약 그룹 정보

- `schedule_reservations.db`: 스케줄 예약 저장
  - `scheduled_reservations` 테이블: 스케줄 정보
  - `reservations` 테이블: 스케줄에서 생성된 예약
